# Installation
Run setup.bat

# Usage

```sh
"-i", "--ip",# Host ip
"-p", "--port", # Port
"-c", "--choice" # UDP(y/n)
"-t", "--times" # default=50000, Packets per one connection
"-th", "--threads" # default=5, Threads. (Higher you go, higher the power.. sort of)
```
